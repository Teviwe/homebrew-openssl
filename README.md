# Local Openssl

## How do I install these formulae?

`brew install local/openssl/<formula>`

Or `brew tap local/openssl` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "local/openssl"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
